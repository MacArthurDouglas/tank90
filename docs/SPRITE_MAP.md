# 坦克大战 精灵映射 (GeneralSprites.png)

- 图集尺寸 400×256，按 **16×16** 网格切片，**PPU=16**（1 格 = 1 世界单位），Point 滤波 / 无压缩。
- 命名规则：`gs_c{列}_r{行}`，列 0–24（左→右），行 0–15（**上→下**）。空白格已跳过。
- 子弹为额外追加的紧贴切片（名字见下），共 354 个精灵。

## 坦克区（列 0–15）
四个颜色块，每块 8×8 格。**每行 = 一辆坦克**，8 列依次为：
`上f0 上f1 | 左f0 左f1 | 下f0 下f1 | 右f0 右f1`（方向序 上/左/下/右，每方向 2 帧动画）。

| 颜色块 | 列范围 | 行范围 | 用途 |
|---|---|---|---|
| 黄 | c0–c7 | r0–r7 | 玩家1 (P1) |
| 白/银 | c8–c15 | r0–r7 | 敌方坦克 |
| 绿 | c0–c7 | r8–r15 | 玩家2 (P2) |
| 紫/红 | c8–c15 | r8–r15 | 敌方坦克(变体/护甲闪烁) |

> ⚠️ 各行对应的「玩家升级等级 / 敌人类型」尚需在 Phase 2 渲染验证。
> 暂定：黄块 r0,r1,r2,r3 = P1 等级1–4；白块 r0–r3 = 敌人 Basic/Fast/Power/Armor。

取某方向首帧的相对列偏移（块内）：上=+0, 左=+2, 下=+4, 右=+6（第二帧再 +1）。
